format
ブジェクトに__format__メソッドを実装した時に、formatメソッドを使った場合、__str__の代わりに使用できます。

HtmlDiff
このクラスは、二つのテキストを左右に並べて比較し、行間あるいは行内の変更点を強調表示するような HTML テーブル (またはテーブルの入った完全な HTML ファイル) を生成するために使います。テーブルは完全差分モード、コンテキスト差分モードのいずれでも生成できます。

__init__
HtmlDiff のインスタンスを初期化します。

make_file
fromlines と tolines (いずれも文字列のリスト) を比較し、行間または行内の変更点が強調表示された行差分の入った表を持つ完全な HTML ファイルを文字列で返します。

ndiff
a と b (文字列のリスト) を比較し、差分 (差分形式の行を生成する(generator)) を、 Differ のスタイルで返します。

restore
差分を生成した元の二つのシーケンスのうち一つを返します。

set_seqs
比較される2つの文字列を設定します。

get_grouped_opcodes
最大 n 行までのコンテクストを含むグループを生成するような、ジェネレータ(generator)を返します。
このメソッドは、 get_opcodes() で返されるグループの中から、似たような差異のかたまりに分け、間に挟まっている変更の無い部分を省きます。
グループは get_opcodes() と同じ書式で返されます。

compare
文字列からなる2つのシーケンスを比較し、差分（を表す文字列からなるシーケンス）を生成します。

Differ
オプションのキーワードパラメータ linejunk と charjunk は、フィルタ関数を渡します
